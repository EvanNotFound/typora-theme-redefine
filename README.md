# Typora Theme Redefine

Enhance your writing experience with the official Typora theme from [Hexo Theme Redefine](https://github.com/EvanNotFound/hexo-theme-redefine). Enjoy a real-time preview of your content in Typora and immerse yourself in the writing process.

## Installation

1. Download the Redefine Typora Theme source code from the [release page](https://github.com/EvanNotFound/typora-theme-redefine/releases).

   The downloaded files should look like this:

   ```
   .
   └── typora-theme-redefine/
       ├── redefine-light.css
       ├── redefine-light-se.css
       ├── redefine-dark.css
       ├── README.md
       ├── LICENSE
       └── redefine/
           └── .....
   ```

2. Open Typora and go to `Preferences` -> `Appearance`.

   Then, click `Open Theme Folder`.

3. Copy all `.css` files and the `redefine/` folder into the theme folder.

4. Restart Typora, and enable the Redefine theme in the `Themes` menu.

5. Set the Light Theme to `Redefine Light` and the Dark Theme to `Redefine Dark`.
